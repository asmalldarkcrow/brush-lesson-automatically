# brush-lesson-automatically
用于学习通刷课，能够全自动化的刷课，无需人为操作。
因为是自动化项目，如果学习通进行了大改，这个项目可能就会失效。
本项目创建时间:2024.5.1(不装了我就是卷狗)
最后更新维护时间:2024.5.1
+++++++++++++++++++++++
项目环境:
python安装pyaibote框架,
配置好aibote环境(框架比较新，跟我这个项目时间差距不大)
安装雷电模拟器，内部安装学习通。（如果你愿意用手机，可以开发者模式接USB操控手机）
模拟器（或手机）安装好aibote的apk文件。
后台清除学习通任务，然后执行脚本
++++++++++++++++++++++++
项目介绍:
支持输入选择要刷的课
支持输入从第几节开始刷
